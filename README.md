# API REST Node.js + Express + Prisma

API desenvolvida com Node.js utilizando Express para gerenciamento de rotas e Prisma ORM para integração com banco de dados MongoDB.

---

## 🚀 Tecnologias
- Node.js
- Express.js
- Prisma ORM
- MongoDB
